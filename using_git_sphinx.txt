Génération de la doc sphinx :

aptitude install sphinx-doc  python-sphinx
cd /doc.
Il doit y avoir un répertoire build, un source, un fichier make.bat et Makefile.

$ make html


Utilisation de git :

Créer un nouveau dépôt sur github.
    - Choisir un gitignore adapté si besoins ainsi que la licence.
    - Copier l'url du repo créé
en local :
    - git clone url du dépot
    - code…
    - git add . (pour ajouter les nouveaux fichiers)
    - git commit -m "init" (créé un nouveau commit en local, avec le commentaire "init")
    - git push (pour uploader les changements sur le repo)
    - git pull (pour récupérer les changements depuis le repo)
    - git status (pour avoir quelques infos sur l'état actuel)


    Ajouter une branche dev:
        - git branch dev (crée la branche)
        - git checkout dev (bascule vers la branche créée)
    
    Merger la branche dev avec master:
        - git checkout master (bascule vers la branche master)
        - git merge dev
        - git status
        - git push
