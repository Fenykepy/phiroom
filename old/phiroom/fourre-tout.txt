===== Les buts de phiroom =====

Phiroom se décompose(ra) en trois parties :

	- Un site public permettant de :

		- Voir des galeries
			- trier les galeries par :
				- ordre de publication (défaut)
				- catégories
				- sous-catégories
				- date de publication
				- mots clés
			- poster un commentaire sous une galerie
			- poster un commentaire concernant une image particulière sous une galerie
			- afficher les images de la galerie dans une lightbox
			- télécharger le fichier original si l'auteur le permet (avec authentification)

		- Voir des images du jour
			- trier les images du jour par :
				- date de publication (défaut)
				- mots clés
			- poster un commentaire sous une image du jour
			- afficher l'image du jour dans une lightbox
			- télécharger le fichier original si l'auteur le permet (avec authentification)

		- Voir des articles
			- trier les articles par :
				- ordre de publication (défaut)
				- catégories
				- sous-catégories
				- date de publication
				- mots clés
			- poster un commentaire sous un article

		- Voir des pages statiques
		- Contacter l'auteur par mail (via un formulaire)
	
	- Un site privé permettant aux utilisateurs inscrits de :
		- poster des commentaires avec un formulaire plus simple
		- télécharger les originaux des images si l'auteur le permet
		- recevoir par mail les notifications de publication d'un nouvel article, galerie ou image du jour
		- recevoir par mail les notifications de réponse à un commentaire
		- utiliser un avatar sur le site (gravatar ou fichier)
		- signaler un commentaire abusif
		- modifier son profil :
			- spécifier une url vers son site perso (en lien avec l'avatar)
			- spécifier une signature pour les commentaires
			- modifier la locale
			- modifier la langue
			- uploader un avatar
			- changer son mot de passe
	
	- Un site privé permettant l'accès à un module bibliothèque aux utilisateurs inscrits et autorisés de :
		- voir les « collections » sur lesquelles ils ont les droits :
			- voir les informations sur chacune des images
			- voir les mots clés
			- effectuer des recherches :
				- par date
				- par notes
				- par mots clés
				- par paramètres de prise de vue

			- voir les images :
				- en grilles
				- en grand
				- à l'échelle 1/1, 
				- en diaporama
		- créer des « collections » personnelles dans un ensemble défini
		- recevoir des messages leur étant destinés
		- s'abonner par mail à leur ensemble de connection

		- Si c'est le webmaster
			- ajouter, supprimer une image d'une collection
			- ajouter, supprimer une image d'une galerie
			- ajouter une collection
			- ajouter une galerie
			- importer des images via http
			- importer des images depuis le disque
			- déplacer les images sur le système de fichier
			- supprimer les images de la base
			- supprimer les images du système de fichier
			- créer des dossiers
			- déplacer des dossiers
			- modifier certaines informations sur les images :
				- les mots clés
				- la date de prise de vue
				- le titre
				- la légende
				- l'auteur
				- le lieu
				- les coordonnées gps
				- la license
	
	- Un site privé servant d'administration permettant de :
		- ajouter, modifier, supprimer des galeries
		- ajouter, modifier, supprimer des images du jour
		- ajouter, modifier, supprimer des articles
		- ajouter, modifier, supprimer des pages statiques
		- ajouter, modifier, supprimer des catégories
		- ajouter, modifier, supprimer des sous-catégories
		- ajouter, modifier, supprimer des utilisateurs
		- ajouter, modifier, supprimer des groupes
		- ajouter, modifier, supprimer des collections
		- ajouter, modifier, supprimer les droits des groupes sur les collections
		- ajouter, modifier, supprimer des images de la base
		- ajouter, supprimer des images sur le serveur
		- modérer les commentaires
		- modifier la configuration :
			- le titre du site
			- le titre de la page d'accueil
			- le sous-titre de la page d'accueil
			- le titre des pages (vu dans l'onglet du navigateur)
			- le logo du site (png)
			- le logo de la page d'accueil
			- la présence ou non de la page d'accueil
			- la pésence ou non d'une page fixe dans le menu
			- le design du site (via des templates)
			- les mails reçus par les administrateurs :
				- commentaires
				- inscriptions
				- modifications de profil
	
			- le nom de domaine du site (pour renseigner les mails)
			- l'affichage des commentaires sur la page d'origine ou sur une page à part
			- le nombre de derniers articles par page
			- le nombre de dernières galeries par page
			- le nombre d'extraits de galeries par page
			- le nombre d'extraits d'articles par page
			- le nombre de derniers articles dans le menu
			- le nombre de dernières galeries dans le menu
			- le nombre de dernières images du jour dans le menu
			- le nombre d'extraits dans le menu
